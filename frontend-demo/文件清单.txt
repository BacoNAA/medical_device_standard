核心页面说明：
-------------
1. ImportPreview.vue (导入预览)
   - 展示Excel导入后的智能处理结果
   - 带颜色标注（绿色/黄色/红色）
   - 新设备确认和待补充信息区域

2. StandardDetail.vue (标准详情)
   - 标准完整信息展示
   - 三个标签页：检验项目、说明性内容、附录
   - 检验项目可展开查看详情

3. StandardList.vue (标准列表)
   - 标准浏览和搜索
   - 点击跳转到详情

4. EquipmentManage.vue (设备管理)
   - 设备库管理
   - 设备关键词库配置

Mock数据说明：
-------------
standardData.js 提供完整的数据结构：
- mockStandardData: 完整标准数据（用于详情页）
- mockImportPreview: 导入预览数据（用于导入页）
- mockStandardList: 标准列表数据
- mockEquipments: 设备信息数据

快速启动：
---------
方式1：双击 "启动开发服务器.bat"
方式2：命令行执行 "npm install && npm run dev"

浏览器访问：http://localhost:3000
