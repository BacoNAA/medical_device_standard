# 医疗器械标准数据库系统 - 前端Demo 项目说明

## 📌 项目概述

本项目是基于《需求规格说明书-医疗器械标准数据库与应用系统_v3.0.md》开发的**纯前端演示系统**，完整实现了需求文档中定义的所有主要功能模块的界面和交互逻辑。

### 开发信息

- **开发时间**: 2025-10-21
- **基于文档**: 需求规格说明书 v3.0
- **系统类型**: 纯前端Demo（无后端）
- **技术栈**: Vue 3 + Vite + Element Plus

## ⚠️ Demo限制说明

### 1. 纯前端实现

- 无后端服务器
- 数据存储在前端状态管理中
- 刷新页面会重置所有操作

### 2. 功能演示

以下功能为演示性质，不生成实际文件：
- Word文档导出
- Excel文件导出

- 不支持真实的数据持久化
- 登录状态保存在localStorage（刷新不丢失）

### 4. 性能优化

- 未做大数据量优化
- 未实现虚拟滚动
- 未做懒加载
