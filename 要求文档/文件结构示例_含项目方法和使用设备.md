## 一、GB标准格式示例（GB 15810-2019 一次性使用无菌注射器）

### 文件清单

```
GB 15810-2019/
├── 标准信息.csv          # 必需
├── 检验要求.csv          # 必需
├── 使用设备.csv          # 可选（提供后减少人工审核）
├── 附录A.csv             # 可选
├── 附录B.csv             # 可选
├── 附录C.csv             # 可选
├── 附录D.csv             # 可选
├── 附录E.csv             # 可选
├── 附录F.csv             # 可选
├── 附录G.csv             # 可选
└── 附录H.csv             # 可选
```

### 1. 标准信息.csv

| 标准编号 | 标准名称 | 英文名称 | 发布日期 | 实施日期 |
|---------|---------|---------|---------|---------|
| GB 15810-2019 | 一次性使用无菌注射器 | Sterile syringes for single use | 2019.10.14 | 2020.11.01 |

### 2. 检验要求.csv

| 项目条款 | 项目名称 | 项目要求 | 注意事项 |
|---------|---------|---------|---------|
| 5.7.1 | 残留容量 | 按附录 B 中 B.2 试验，在活塞被完全推入至外套底端，注射器的最大残留容量应符合表1的要求。 | |
| 5.7.2 | 器身密合性 | 按附录C试验，活塞或密封圈处应无液体泄漏现象。按附录D试验，活塞或密封圈处应无气体泄漏现象，且压力表的读数不应持续下降。 | 注意试验顺序 |
| 5.7.3 | 滑动性能 | 按附录 E 试验，滑动性能力值应符合表2的要求。 | |

**说明：**
- GB标准的项目要求中通常包含附录引用（如"按附录 B 中 B.2 试验"）
- 系统会自动识别这些引用，从对应的附录文件中提取方法内容

### 3. 使用设备.csv（可选）

| 项目条款 | 设备名称 | 设备编号 |
|---------|---------|---------|
| 5.7.1 | 电子天平 | A001 |
| 5.7.2 | 注射器器身密合性测试仪 | B001 |
| 5.7.2 | 真空泵 | B002 |
| 5.7.3 | 滑动性能测试仪 | C001 |

**说明：**
- 此文件为可选文件
- 如提供此文件，系统将直接使用，减少智能识别的工作量
- 如不提供，系统将从附录内容中智能识别设备

### 4. 附录B.csv

| 章节编号 | 章节标题 | 章节内容 | 使用设备 |
|---------|---------|---------|---------|
| B.1 | 容量允差 | 注射器抽吸水至刻度容量，称取排尽后水的质量，排出体积与刻度容量之差为容量允差。使用电子天平，精度为0.1 mg... | 电子天平 |
| B.2 | 残留容量测定 | 以同一支注射器，抽吸前与排空后注射器的质量之差，计算出注射器的残留容量。使用电子天平，精度为0.1 mg。打开注射器的初包装，用电子天平称取注射器的质量... | 电子天平 |

## 二、YY标准格式示例（YY 0469-2023 医用外科口罩）

### 文件清单

```
YY 0469-2023/
├── 标准信息.csv          # 必需
├── 检验要求.csv          # 必需
├── 项目方法.csv          # 必需（YY标准格式特有）
├── 使用设备.csv          # 可选
└── 附录A.csv             # 可选（如有引用）
```

### 1. 标准信息.csv

| 标准编号 | 标准名称 | 发布日期 | 实施日期 |
|---------|---------|---------|---------|
| YY 0469-2023 | 医用外科口罩 | 2023.03.17 | 2024.04.01 |

### 2. 检验要求.csv

| 项目条款 | 项目名称 | 项目要求 | 注意事项 |
|---------|---------|---------|---------|
| 5.1 | 外观 | 口罩外观应整洁、形状完好，表面不得有破损、污渍；口罩各部件连接处应牢固。 | 目视检查 |
| 5.2 | 结构与尺寸 | 口罩结构应符合4.1的要求。口罩基本尺寸应符合表1的规定。 | |
| 5.3 | 鼻夹 | 口罩上应配有鼻夹，鼻夹由可塑性材料制成，长度应不小于8.0 cm。 | |
| 5.4 | 口罩带 | 口罩带应有足够的强度，固定牢固，每根口罩带与口罩体连接点处的断裂强力应不小于10 N。 | |

**说明：**
- YY标准的项目要求通常不直接引用附录
- 方法内容在单独的"项目方法.csv"文件中，与要求一一对应

### 3. 项目方法.csv（YY标准格式必需）

| 项目条款 | 项目方法 | 方法引用 |
|---------|---------|---------|
| 5.1 | 在自然光线下，目测检查口罩外观。检查口罩表面是否有破损、污渍，各部件连接是否牢固。每批次抽样20只，全部合格则判定通过。 | |
| 5.2 | 使用直尺或卷尺测量口罩长度和宽度，精度为1 mm。测量位置应在口罩平铺状态下，长度为口罩上下边缘的最大距离，宽度为左右边缘的最大距离。测量10只样品，计算平均值。 | |
| 5.3 | 使用直尺测量鼻夹长度，精度为1 mm。用手弯折鼻夹，检查其可塑性。测量10只样品。 | |
| 5.4 | 按GB/T 19083-2010中附录A规定的方法进行试验。使用拉力试验机，以(100±20) mm/min的速度拉伸口罩带，直至口罩带与口罩体连接处断裂，记录断裂时的最大力值。测量5个连接点。 | 见GB/T 19083-2010附录A |

**说明：**
- 项目条款与检验要求.csv中的项目条款一一对应
- 方法引用字段可选，如引用了其他标准的附录可在此注明

### 4. 使用设备.csv（可选）

| 项目条款 | 设备名称 | 设备编号 |
|---------|---------|---------|
| 5.2 | 直尺 | D001 |
| 5.2 | 卷尺 | D002 |
| 5.3 | 直尺 | D001 |
| 5.4 | 拉力试验机 | E001 |

## 三、系统处理逻辑对比

### GB标准格式（附录引用型）

```
处理流程：
1. 读取检验要求.csv → "按附录 B 中 B.2 试验"
2. 正则匹配 → 识别"附录B"、"B.2"
3. 打开附录B.csv → 查找章节编号"B.2"
4. 提取B.2的章节内容 → 填入MethodContent字段
5. 设置MethodType=2（引用附录）
6. 如提供使用设备.csv：直接匹配设备
7. 如未提供使用设备.csv：从附录内容智能识别设备
```

### YY标准格式（一一对应型）

```
处理流程：
1. 读取检验要求.csv → 项目条款"5.4"
2. 打开项目方法.csv → 查找项目条款"5.4"
3. 提取项目方法内容 → 填入MethodContent字段
4. 设置MethodType=3（完整方法内容）
5. 如提供使用设备.csv：直接匹配设备
6. 如未提供使用设备.csv：从项目方法内容智能识别设备
```

## 四、数据输入员准备文件指南

### GB标准（如GB 15810-2019）

**必需文件：**
1. 标准信息.csv
2. 检验要求.csv

**推荐文件：**
3. 使用设备.csv（提供后减少80%人工审核工作）
4. 附录X.csv（有附录引用时必需）

**不需要文件：**
- ❌ 项目方法.csv（GB标准不需要，方法从附录自动提取）

### YY标准（如YY 0469-2023）

**必需文件：**
1. 标准信息.csv
2. 检验要求.csv
3. 项目方法.csv（✅ YY标准必需）

**推荐文件：**
4. 使用设备.csv（提供后减少80%人工审核工作）

**可选文件：**
5. 附录X.csv（如项目方法中引用了附录）

## 五、优势总结

### 1. 兼容两种标准格式

- **GB标准**：通过附录引用自动提取方法
- **YY标准**：通过项目方法文件直接对应

### 2. 减少人工工作量

| 操作 | 原方式 | 新方式（提供使用设备.csv） | 新方式（不提供） |
|------|--------|--------------------------|----------------|
| 填写项目方法 | 100%人工 | 0%（系统自动） | 0%（系统自动） |
| 填写使用设备 | 100%人工 | 0%（系统自动） | 20%人工微调 |
| 人工操作量 | 100% | **0%** | **20%** |

### 3. 灵活性强

- **完整提供**：标准信息 + 检验要求 + 项目方法 + 使用设备 → 100%自动化
- **部分提供**：标准信息 + 检验要求 + 项目方法 → 80%自动化（需微调设备）
- **最少提供**：标准信息 + 检验要求 → 60%自动化（需微调方法和设备）
